Afin de pouvoir utiliser cette librairie, vous devez avant ajouter dans votre projet ionic les dependances suivantes :

[source, json]
package.json
----
"@ionic-native/native-storage": "^5.27.0",
"@ionic/storage": "^2.2.0"
----
